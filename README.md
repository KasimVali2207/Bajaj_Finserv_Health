## Objective
A REST API (`POST /bfhl`) that processes an array of data and returns:
- Even/Odd numbers
- Alphabets in uppercase
- Special characters
- Sum of numbers
- Concatenated alphabets (reverse + alternating caps)

---

## Example Request
```json
{
  "data": ["a", "1", "334", "4", "R", "$"]
}
